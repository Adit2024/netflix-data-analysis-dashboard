Netflix Data Analysis Dashboard 📊

This project presents an interactive Netflix data dashboard built using Microsoft Power BI. The dashboard analyzes Netflix movies and TV shows to uncover insights about content distribution, ratings, genres, and global availability.

The goal of this project is to demonstrate data analysis, visualization, and business intelligence skills by transforming raw Netflix dataset into meaningful insights.

Key Dashboard Metrics

Total Shows: 8803

Total Ratings Categories: 19

Total Genres: 515

Total Directors: 4527

Content Timeline: 1925 – 2021

Dashboard Features

📈 Rating Analysis
Shows distribution of Netflix content based on ratings such as TV-MA, TV-14, TV-PG, etc.

🎬 Genre Distribution
Identifies the most popular genres including Drama, Documentary, Stand-Up Comedy, and Kids content.

🌍 Country-wise Content Distribution
A world map visualization showing which countries produce the most Netflix content.

🍿 Movies vs TV Shows Comparison
A donut chart comparing the proportion of movies and TV shows available on Netflix.

📅 Content Growth Over Time
A trend analysis showing how Netflix content has increased over the years.

Project Insights

Netflix contains significantly more Movies than TV Shows.

TV-MA and TV-14 ratings dominate the platform.

Drama and Documentary are among the most common genres.

Content growth increased rapidly after 2015, showing Netflix's expansion phase.
